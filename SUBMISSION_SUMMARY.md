# Submission Summary

## Fiber Reliability Kit

**Category:** Node, Routing, Cross-Chain, and Diagnostics Infrastructure

Fiber Reliability Kit is a reusable reliability layer for Fiber Network on Nervos CKB. It focuses on deterministic diagnosis and recovery guidance rather than wallet custody, installers, or generic dashboards.

## Deliverables

- `@fiber-reliability/sdk`
- `fiber-doctor` CLI
- Reliability Lab scaffold

## What it demonstrates

- payment intent analysis
- readiness diagnostics
- payment failure classification
- CCH inspection warnings
- dry-run rebalance planning
- fixture-first demo and CLI workflows

## Current live/mocked split

- Fully working: SDK fixture diagnostics, CLI fixture mode, Reliability Lab scaffold
- Optional live mode: local/operator-provided Fiber RPC
- Mocked/simulated: public hosted demo data
